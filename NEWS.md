# EpigenicR 0.1.0

## Initial Release (2026-03-06)

This is the first official release of EpigenicR, a comprehensive R package for epigenomic data analysis and visualization developed for the EpiFinder platform.

### Core Features

* **Metadata Management**: Extract and organize sample information from BigWig filenames with `create_metadata_df()`
* **Quality Control**: Visualize QC statistics across markers, conditions, and replicates with `plot_qc_stats()`
* **Annotation Retrieval**: Download and process GTF/BED files for genomic features with `ensure_gtf_and_beds()`
* **ChromHMM Integration**: Retrieve chromatin state annotations from Epigenome Roadmap with `download_chromhmm_annotations()`
* **EPK Objects**: Create EPK (EpiPeaK) objects containing MultiAssayExperiment data structures
* **Correlation Analysis**: Compute sample-sample correlations across markers with `compute_sample_cor()` and `compute_all_cor()`

### Example Datasets

Complete toy dataset included for testing and examples:

* **6 BigWig files**: Real chromatin (H3K4me3, H3K27ac, INPUT) and methylation (5mC) profiles from two samples (SAMPLE-0008, SAMPLE-0054)
* **toy_metadata**: Pre-extracted metadata from filenames (6 files)
* **toy_stats_summary**: QC statistics with read counts, duplication rates, and library sizes (8 samples)
* **toy_genes**: Example gene coordinates on chr22 for testing (5 genes); to show the genomic ranges format only  
* **enrichment_results**: Chromatin state enrichment data for 3 markers (H3K4me3, H3K27ac, 5mC)
* **profile_results**: Signal enrichment profiles around TSS and CpG islands
* **project.epk.rds**: Example EPK object demonstrating the final data structure

All toy data loadable with standard `data()` function.

### Dependencies

* Core dependencies: dplyr, tidyr, ggplot2, wigglescout
* Genomic data structures: GenomicRanges, SummarizedExperiment, MultiAssayExperiment
* Requires R >= 4.1.0
* Special note: Requires future v1.34.0 for wigglescout compatibility

### Installation

```r
# Install future version 1.34.0
packageurl <- "http://cran.r-project.org/src/contrib/Archive/future/future_1.34.0.tar.gz"
install.packages(packageurl, repos=NULL, type="source")

# Install wigglescout from GitHub
remotes::install_github('cnluzon/wigglescout', build_vignettes = TRUE, force = TRUE)

# Install EpigenicR
remotes::install_github("epigenica/EpigenicR")
```
